# TaskManager-pyqt4
A task manager for Linux

Enviorments
 - Python2.6 or above
 - PyQt4 with Qwt5
 - Numpy

Run
 - python TaskMgr.py 

or 
 - python2 TaskMgr.py

if you set Python3 as your default python command
